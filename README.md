# Calculadora de Subnetting

Calculadora de Subnetting interactiva en HTML, CSS y JS. Permite calcular subredes, hosts, broadcast, máscaras y representación binaria, con diseño moderno dark y blur.

## Características
- Validación de IP y CIDR en tiempo real.  
- Resultados detallados: Dirección de red, máscara, hosts, broadcast, wildcard, primeros y últimos hosts.  
- Representación binaria de IP, máscara, red y broadcast.  
- Tabla de referencia CIDR real con paginación y colapsable.  
- Diseño dark moderno con blur difuminado y fuente Hack Nerd Font.  
- Toast elegante al copiar resultados.  
- Resultados alineados debajo de los títulos para mayor legibilidad.  

## Instalación

1. Primera opción:

- Clona el repositorio:

```bash
git clone https://github.com/D1se0/calculadora_subnetting.git
```

2. Segunda opción:

- Abre el `link` en tu navegador desde la `URL` de `GitHub Pages`.

## Uso

- Ingresa una IP y CIDR o IP y máscara (`192.168.1.0/24` o `192.168.1.0 255.255.255.0`).
- Verás los badges de clase, tipo de red y CIDR, los resultados en columnas y la representación binaria.
- Copia cualquier resultado haciendo clic en el icono de copiar; aparecerá un mensaje de confirmación.

## Créditos

- Desarrollado con `HTML`, `CSS` y `JavaScript` puro.
- Fuente `Hack Nerd Font` incluida para visualización de binarios y estilo moderno.
